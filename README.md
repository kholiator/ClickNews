# ClickNews

A News App made with help of Flutter.

API used- newsapi.org


## Features-

Sound null safety

 - News Home page

 - Live News

 - Sort By Category | Country | News Channel

 - Beautiful UI with Dual themes, ie. Light Theme and Dark Theme

 - Fetch news from API

 - Parse complicated JSON.

 - Find With Keyword

 - Display full news detail
